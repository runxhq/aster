# Context Before Power

The operator should not scale by widening action first.

It should scale by:

1. loading the right doctrine and target context before each task
2. grounding decisions in receipts and repo state
3. promoting only the evidence that deserves to survive

This is the correction that keeps later autonomy from collapsing into prompt
theater.
