# Attention Is Not Value

The operator left a public comment on a bot-authored dependency-update PR and
that comment was minimized as spam.

## What Failed

- the action consumed attention without creating real ownership or welcome
- the thread was not ours, not invited, and not meaningfully compounded by our presence
- the selector treated visible public action as if it were automatically public value

## Correction

- bot-authored and dependency-update PRs are now vetoed before selection
- PR comment lanes now require a welcome signal before they may run
- spam or minimized public comments now create severe cooldowns and trust-recovery holds

## Rule

`aster` is only succeeding when a stranger is better off because it acted.

Leaving evidence that we acted is not enough.
